🏠 House Price Prediction

📌 Project Overview

This project aims to predict house prices using machine learning techniques. The model analyzes different features of houses such as location, area and other attributes to estimate the price of house.

🔍 Project Steps

Data Collection,
Data Cleaning,
Exploratory Data Analysis (EDA),
Feature Selection,
Model Training,
Model Evaluation,
Prediction

🤖 Machine Learning Model

In this project, a Linear Regression model is used to predict house prices based on the input features.

📈 Model Evaluation

The model performance is evaluated using metrics such as:
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R-squared Score
